# Meditation App (Flask + HTML/CSS/JS)

## Run
1. Install dependencies:
   ```bash
   pip install flask
   ```
2. Start server:
   ```bash
   python app.py
   ```
3. Open: http://127.0.0.1:5000/

## What it does
- Breathing guide (inhale/hold/exhale) with animated orb
- Session timer (start/pause/reset)
- Saves completed sessions to `data/sessions.json`
